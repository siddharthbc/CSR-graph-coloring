Algorithm 2 — Greedy List-Coloring of Conflict Graph

From Picasso: Memory-Efficient Graph Coloring Using Palettes (Ferdous et al.)
Processes vertices in order of smallest current color-list size (bucket structure). When a vertex is colored, its chosen color is removed from all neighbors' lists.

Pseudocode (Algorithm 2)
B ← Array of bucket lists
// Create initial buckets
for v ∈ V do Insert v to B[colList(v).size()]
Mark all vertices as unprocessed
Vu ← ∅
while ∃ unprocessed vertex do
  Pick vertex v from lowest bucket
  Color v from colList(v), say color c
  Remove v from bucket; mark processed
  for u ∈ adj(v) do
    if u uncolored and c ∈ colList(u)
      Remove c from colList(u)
      if colList(u) empty → Vu ∪= u
      Move u to B[colList(u).size()]
