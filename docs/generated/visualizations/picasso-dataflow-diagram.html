Max Boundary/PE = most boundary edges any single PE must send per round. Max Relay Through/PE = most pass-through wavelets any PE must relay per round. As PE count grows, local work decreases but relay traffic increases — the fundamental communication/computation trade-off.

Key Data Transfers Across the Software ↔ Hardware Boundary

Direction Data Format Per PE
H2D ↓ csr_offsets int32[max_local_verts+1] CSR row pointers
csr_adj int32[max_local_edges] CSR adjacency (global IDs)
colors int32[max_local_verts] Initialized to −1
global_vertex_ids int32[max_local_verts] GID for each local vertex
boundary_* (3 arrays) int32[max_boundary] × 3 local_idx, neighbor_gid, direction
verts_per_pe int32[1] Chunk size for GID→PE routing
num_boundary int32[1] Active boundary entries
expected_total_recv int32[1] Precomputed wavelet count
D2H ↑ colors int32[max_local_verts] Final color assignment
coloring_timer f32[3] Packed TSC timestamps
perf_counters int32[7] rounds, sends, recvs, relays, ...
