# Generated Docs Index

Purpose: hold generated or presentation-oriented artifacts that are useful to keep, but are not primary sources of implementation truth.

## Layout

- `reports/` — generated or duplicate PDF outputs
- `visualizations/` — generated HTML visualizations and explainers

### Report PDF Status

- `reports/Picasso_Implementation_End_to_End.pdf` — duplicate generated render; canonical linked copy remains under `docs/`
- `reports/Picasso_Technical_Deep_Dive.pdf` — generated render of the active TeX source

- `reports/undecided/` — report PDFs kept but not yet classified as canonical or redundant

## Current Policy

- Keep canonical implementation docs under `docs/active/` or root control files.
- Keep historical notes under `docs/archive/`.
- Keep generated visual material and duplicate report renders here.
